# 🎸 Блюз-инструмент 2

Развитие [блюз-инструмента](https://gromozeka1980.github.io/blues-instrument/) для интуитивной импровизации. Работает на компьютере, планшете и телефоне — мышь, клавиатура и тач.

**▶ Играть: https://gromozeka1980.github.io/blues-instrument-2/**

## Идея

Клавиши задают **не ноту, а шаг по гамме** относительно предыдущей: `0` — повтор, `+1` — на ступень выше, `−2` — на две ниже. Гамма заблокирована, поэтому промахнуться мимо «блюзового» звука невозможно.

## Что нового по сравнению с первой версией

- **Лад:** Минор / Мажор / Микс — две блюзовые гаммы (минорная и мажорная) и их объединение. Роли нот и цели бендов пересчитываются автоматически.
- **Бэк-трек:** Мажорный блюз / Минорный блюз / Дрон-вамп — переключается на лету.

## Как играть

- **Нижний ряд** (большие клавиши) — частые ходы `−2 −1 0 +1 +2`.
- **Верхний ряд** (поменьше) — прыжки `−5 … +5`.
- **Бенд:** тяни клавишу вверх мышью/пальцем или держи **Shift**. Нота подтягивается, как струна, ровно до нужного тона — перетянуть нельзя. Ноты, которые гнуть не стоит, не поддаются.

Клавиатура: `S D F G H · Пробел · J K L ; '` · `Shift` — бенд · `[` `]` — октава · `Backspace` — к тонике.

Диапазон ограничен A1…A6.

## Технологии

Один файл `index.html`, Web Audio API, без зависимостей и сборки.
